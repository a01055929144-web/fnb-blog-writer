# F&B 지역 특화 블로그 자동 작성기

수도권 식자재 중개 영업을 위한 네이버 블로그 SEO 포스팅 자동 작성 도구입니다.

## 주요 기능

- 수도권 31개 지역 상권 특성 데이터 내장 (서울 15개 + 경기 11개 + 인천 5개)
- 업종별 일반 데이터 (계절 시세·트렌드·원가절감 팁·품목별 단가)
- 롱테일 키워드 자동 추출 (지역 + 품목 + 의도 조합)
- Claude API 연동 1500-2000자 상세형 포스팅 자동 생성
- 구글 시트 자동 저장
- 고정 상담 채널 CTA 자동 삽입

## 사용 방법

1. Claude API 키 입력
2. 업종 / 지역 / 품목 선택
3. 롱테일 키워드 추출
4. 포스팅 생성 → 복사 → 네이버 블로그 발행
