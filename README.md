# F&B 지역 특화 블로그 자동 작성기

수도권 식자재 중개 영업을 위한 네이버 블로그 SEO 포스팅 자동 작성 도구입니다.

## 주요 기능

- 수도권 **73개 지역** 상권 특성 데이터 내장
  - 서울 25개 구
  - 경기 38개 (수원·성남·고양·용인·안양·안산 군구 분리 포함)
  - 인천 10개 구·군
- 업종별 일반 데이터 (12개월 계절 시세·트렌드·원가절감 팁·품목별 단가)
- 데이터 레이어 토글 (일반 데이터 / 지역 특성 데이터 ON·OFF)
- 롱테일 키워드 자동 추출 (지역 + 품목 + 의도 조합, 경쟁도 분류)
- Claude API 연동 1500-2000자 상세형 포스팅 자동 생성
- Google Apps Script 연동 구글 시트 자동 저장
- 고정 상담 채널 CTA 자동 삽입 (카카오 오픈채팅 · 네이버 블로그 · 전화)

## 사용 방법

### 1. 초기 설정
1. 페이지 상단 **API 설정** 배너에 Anthropic API 키 입력 → 저장
2. Google Apps Script Web App URL 입력 → 저장 (아래 시트 연동 참고)
3. 왼쪽 사이드바에서 전화번호 입력 → 저장

### 2. 포스팅 생성
1. 업종 선택 (한식/축산·일식/수산·중식 등 8개)
2. 수도권 지역 선택 (서울·경기·인천 탭, 최대 5개)
3. 세부 품목 선택 (최대 5개)
4. 콘텐츠 타입 체크 (정보성 SEO·원가절감·대체 식자재 등)
5. 데이터 레이어 설정 (일반 데이터 + 지역 특성 데이터)
6. **롱테일 키워드 추출** 클릭
7. 원하는 키워드 선택
8. **포스팅 생성** 클릭 → 자동으로 구글 시트 저장

### 3. 발행
- 결과 카드에서 **복사** 클릭
- 네이버 블로그에 붙여넣기 → 발행

## Google Apps Script 시트 연동 설정

1. [Google Apps Script](https://script.google.com) 접속
2. **새 프로젝트** 생성
3. `Code.gs` 파일 내용 전체 붙여넣기
4. `SHEET_ID` 값 확인 (이미 설정됨)
5. **배포** → **웹 앱으로 배포**
   - 실행 계정: 나
   - 액세스 권한: **모든 사용자**
6. 발급된 URL을 블로그 작성기 상단 **시트 URL** 입력창에 붙여넣기
